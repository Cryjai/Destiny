## ✦ 關於命途

命途是一個結合**傳統中華命理**與**現代 UI 設計**的命運工具 App。
不是迷信，是系統——幾千年的人生觀察，壓縮成你掌心的工具。

> 「知命者不怨天，知己者不怨人。」— 荀子

## ✦ 功能

| 模組 | 說明 |
|------|------|
| 🀄 八字四柱 | 根據出生年月日時推算命盤，解讀五行格局 |
| ⭐ 紫微斗數 | 十四主星命盤排佈，揭示人生走勢 |
| ☯️ 易經占卜 | 六十四卦演算，指引當下困惑 |
| 🎴 每日抽 | 每日一抽，獲取今日運勢提示 |

## ✦ 快速開始

### 📱 Android APK（直接安裝）
1. 下載 [`destiny-destiny.apk`](https://github.com/Cryjai/Destiny/raw/main/destiny-destiny.apk)
2. 允許「未知來源安裝」
3. 開啟 App，輸入出生資料即用

### 🌐 Web 版（免安裝）
直接開啟：[https://cryjai.github.io/Destiny/](https://cryjai.github.io/Destiny/)
